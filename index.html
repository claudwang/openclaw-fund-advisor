🚀 一键安装

只需要一条命令，即可完成整个 Skill 的安装！
自动克隆、自动配置、立即可用。

🔧 在终端中运行以下命令：

bash -c "$(curl -s https://raw.githubusercontent.com/claudwang/openclaw-fund-advisor/main/install.sh)"

📋 安装步骤：

1. 复制上面的安装命令
2. 在终端中粘贴并运行
3. 等待安装完成提示
4. 编辑持仓文件：~/.openclaw/skills/fund-advisor/references/portfolio.md
5. 开始使用！

📝 配置持仓

安装后，编辑持仓文件即可获得个性化建议：

nano ~/.openclaw/skills/fund-advisor/references/portfolio.md
